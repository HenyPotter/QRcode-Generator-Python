# QR code generator
## Description

QR codes are very useful, so lets generate one

### What to do


  - Install pyqrcode
  - Install png
  - Run the file


### Intalling instaloader

Open cmd and type there this:

```
pip install pyqrcode
```

### Installing png

Open cmd and type there this:
```
pip install pypng
```
### Running the file

Open cmd and type there this:
```
python3 path of your project
```
#### Example
```
python3 C:Users/GitHub/Projects/main.py
```
